### Plotly Dash – Web-App zur Analyse und Steuerung eines Messsystems

Im Rahmen eines eigenständigen Projekts habe ich mit **Plotly Dash** eine datengetriebene Webanwendung entwickelt, die zwei externe Schnittstellen integriert:  
eine **Zeiss PiWebAPI** sowie einen **Microsoft SQL Server**. Die App visualisiert, analysiert und manipuliert die Daten beider Quellen interaktiv im Webinterface.

Zur Authentifizierung verwende ich **Flask-Login**, für das Deployment einen **WSGI-Server (Gunicorn)** und zur Datenspeicherung eine **MongoDB**. Die gesamte Anwendung läuft als **monolithischer Docker-Container** auf einem internen **SUSE Linux Server**.  

Mithilfe von **GitLab CI/CD** wird die App automatisch gebaut und deployed. Ein integriertes Logging ermöglicht die Nachverfolgbarkeit von Aktionen und Fehlern. Die Anwendung kann Daten exportieren, die anschließend zur **Bestimmung der Prozessfähigkeit eines Messsystems** genutzt werden.

**Technologien:** Plotly Dash, Flask, MSSQL, PiWebAPI, MongoDB, Docker, Gunicorn, GitLab CI/CD, Linux (SLES)
